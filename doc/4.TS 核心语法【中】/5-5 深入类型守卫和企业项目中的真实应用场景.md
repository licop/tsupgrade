# 5-5 深入类型守卫和企业项目中的真实应用场景

　　**为什么要用类型守卫：**

　　**类型守卫定义：** 在 语句的块级作用域【 if 语句内或条目运算符表达式内】缩小变量的一种类型推断的行为。

　　**类型守卫产生时机**：TS  条件语句中遇到下列条件关键字时，会在语句的块级作用域内缩小变量的类型，这种类型推断的行为称作类型守卫 ( Type Guard )。类型守卫可以帮助我们在块级作用域中获得更为需要的精确变量类型。

* **实例判断：**`instanceof`
* **属性或者方法判断：**`in`
* **类型判断：**`typeof`
* **字面量相等判断：**`==`, `===`, `!=`, `!==`
